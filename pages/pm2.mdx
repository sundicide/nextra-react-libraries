# Template

## Using ecosystem file

1. make `pm2-ecosystem.json` file
    ```json
    {
      "apps": [
        {
          "name": "lib-samples",
          "script": "npm",
          "args" : "start"
        }
      ]
    }
    ```
2. add script `package.json` <br />
    `"deploy:pm2": "pm2 start pm2-ecosystem.json"`

## Using cli

```shell
pm2 start --name operation npm -- start
```
