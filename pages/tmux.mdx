# Tmux

## On Tmux
```bash
C^b + $ # rename session
C^b + % # split pane
C^b + d # detach
C^b + s # list all session
C^b + -> / <-  # move cursor to next panel
```

## Out of Tmux

```bash
tmux a # attach last session
```

## Settings

* enable mouse scrolling
  1. vi `~/.tmux.conf`
  2. insert line `set -g mouse on`
